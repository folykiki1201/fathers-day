父亲节信封互动版

打开方式：
直接打开 index.html 即可预览。

照片位置：
assets/photo-01.jpg
assets/photo-02.jpg
assets/photo-03.jpg
assets/photo-04.jpg

Procreate 信件素材：
把透明背景或白底 PNG 放进 assets 文件夹，命名为 letter.png。
页面打开信封后，中间会优先显示 letter.png。

建议：
letter.png 最好是竖版，比例接近 3:4 或 A5 竖版。
如果你想放手写标题、贴纸、小花，也可以先放进 assets 文件夹，之后告诉我文件名，我再帮你融入页面。

旧版保留：
原来的全屏故事版仍在 outputs/fathers-day-story，没有删除。
